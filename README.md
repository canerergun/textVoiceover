# 🎙️ Emel TTS - Modern Türkçe Seslendirme

Modern ve sade arayüze sahip, **yalnızca Emel Neural** sesini kullanan kaliteli bir Türkçe metinden sese (TTS) uygulaması.

Microsoft Edge TTS altyapısını kullanarak doğal ve akıcı Türkçe seslendirme yapmanızı sağlar.









---

## ✨ Özellikler

- **Sadece Emel Neural** sesi (en kaliteli Türkçe ses)
- Modern ve temiz arayüz tasarımı
- Gerçek zamanlı **hız ayarı** (50% – 200%)
- Hızlı preset butonları: *Yavaş, Normal, Hızlı*
- **Direkt MP3 olarak kaydetme** (dinlemeden kaydetme özelliği)
- Ses dinleme ve durdurma kontrolleri
- Karakter ve kelime sayacı
- Metin dosyasından yükleme desteği
- Örnek metin butonu
- Geçici dosyaları otomatik temizleme

---

## 🛠️ Kurulum

### Gereksinimler

- Python 3.8 veya üstü

### Kütüphanelerin Kurulumu

```bash
pip install edge-tts pygame
```

🚀 Kullanım
Projeyi indirin veya klonlayın.
app.py (veya dosyanızın adı) olarak kaydedin.
Terminalde şu komutu çalıştırın:
```bash
python app.py
```


## 📱 Arayüz Özellikleri
- Sol taraf: Geniş metin giriş alanı
- Sağ taraf: Kontrol paneli
- Okuma hızı slider’ı + preset butonları
- "Seslendir" ve "Kaydet" butonları
- "Dinle" ve "Dur" kontrolleri
- Üst kısım: Karakter ve kelime sayacı
- Alt kısım: Durum çubuğu ve ilerleme çubuğu


## 🧪 Kullanılan Teknolojiler
- Tkinter — Arayüz için
- edge-tts — Microsoft Edge’in yüksek kaliteli TTS motoru
- pygame — Ses oynatma
- asyncio + threading — Uygulamanın donmadan çalışması için
- Modern renk paleti ve temiz tipografi

## 📝 Örnek Kullanım
- Metin kutusuna yazın veya örnek metni yükleyin.
- İsterseniz hızı ayarlayın.
- "Seslendir" butonuna basarak dinleyin.
Veya "Kaydet" butonuna basarak direkt MP3 olarak kaydedin.
